# Linux/Mac Configuration

To quickly install the development environment for Linux and Mac OS X.

## Configurations:

This repo includes the following configuration files:

1. vim
1. bash
1. input
1. screen


## System Requirements

### Overall

* vim 7.4 compiled with option `+python`
* python

## Install Command

    ./install
